**Сокращение признакового пространства с помощью генетического алгоритма**

Дипломная работа. Написан генетический алгоритм, осуществляющий выбор значимых признаков. Протестированы различные функции пригодности (accuracy, кросс-энтропия, кросс-энтропия со штрафными функциями).
Алгоритм протестирован на приложенных наборах данных.
